Find picture in picture
========================

Notebook with iterative code to find the best match for a small picture within a
larger picture. This can be used to identify and select layout elements such as
tables within articles.

Basic examples for the first code part are included in this folder. For 
complete examples use the download notebooks provided in the webpage_search_download/ directory of
this repository.

Requiremd packages:
------------------
numpy 
matplotlib 
jupyter
opencv
imutils
