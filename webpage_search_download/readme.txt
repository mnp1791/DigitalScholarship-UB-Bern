complex webarchive example
==========================
Notebooks:
search_download_BZ_example.ipynb
download_BZ_whole_year.ipynb

Example for downloading files from a webarchive and search webcontent within a
complex page structure. It requires an analysis of the webarchive structure for
each case. The first notebook contains code for downloading a selected page for a 
a list of issues, the second notebook downloads all pages of all issues for a given year.
